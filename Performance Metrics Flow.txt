## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- PostgreSQL 15+
- Redis (optional, for caching)

### Installation

```bash
# Clone the repository
git clone https://github.com/rachitmehta/aegis-rag.git
cd aegis-rag

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set environment variables
export DATABASE_URL=postgres://user:pass@localhost:5432/aegis_rag
export OPENAI_API_KEY=your-api-key-here
export DJANGO_SECRET_KEY=your-secret-key

# Run migrations
python manage.py migrate

# Generate synthetic enterprise data
python manage.py generate_synthetic_data

# Build FAISS and BM25 indices
python manage.py build_indices

# Start the server
python manage.py runserver
